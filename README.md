# parcel-react-start-kit

```
git clone https://github.com/satsuya0114/parcel-react-start-kit.git
cd parcel-react-start-kit
npm install
npm start 
```
---

```
npm start
```
-> run at localhost development env ( by parcel-bundler ) 

---

```
npm run build 
``` 
-> run to build a production env at ./dist
